💻 Arduino LED Blinking – Code Implementation

As part of my learning in embedded systems, I implemented a simple LED blinking program using the Arduino IDE and the Arduino Uno.

In this project, I wrote a basic program to control LEDs connected to the Arduino digital pins. The code sends HIGH and LOW signals to the LEDs, creating a blinking effect in a specific pattern.

🔹 Key Learning Points:
• Understanding Arduino programming structure (setup() and loop())
• Using digital output pins to control LEDs
• Writing and testing code in Arduino IDE
• Learning how microcontrollers interact with electronic components

This small project helped me strengthen my understanding of basic programming logic and hardware interaction in embedded systems.
